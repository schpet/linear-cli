# schema

> Print the GraphQL schema to stdout

## Usage

```
Usage:   linear schema
Version: 1.7.0        

Description:

  Print the GraphQL schema to stdout

Options:

  -h, --help            - Show this help.                                     
  --json                - Output as JSON introspection result instead of SDL  
  -o, --output  <file>  - Write schema to file instead of stdout
```
