# auth

> Manage Linear authentication

## Usage

```
Usage:   linear auth
Version: 1.8.0      

Description:

  Manage Linear authentication

Options:

  -h, --help  - Show this help.  

Commands:

  token   - Print the configured API token                
  whoami  - Print information about the authenticated user
```

### token
